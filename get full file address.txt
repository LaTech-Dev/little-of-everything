This is a quick way to copy a file's path without altering the Finder window. 
1.   Open Finder: and find the file you want.
2.   Right-click: (or Control-click) the file.
3.   Hold down the Option key: on your keyboard.
4.   A contextual menu will appear. Select the option that says "Copy [filename] as File Path".
